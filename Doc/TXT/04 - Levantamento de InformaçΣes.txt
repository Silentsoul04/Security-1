Fingerprint
===========

Levantar as informações dos sistemas operacionais.

Para que o fingerprint apresentar resultados confiáveis são necessárias análises complexas, como:
	* Analise de pacotes que trafegam pela rede
	* Leitura de banners (assinaturas do sistema)
	* Análise de particulariedades da pilha TCP/IP


Fingerprint passivo --> scanner de rede para farejar a rede
fingerprint ativo -> o scanner envia pacotes manipulados e forjados.

Fingerprint Passivo
-------------------

p0f para ouvir o trafego de rede

vamos usar o app

p0f -i eth0 -o p0f.log

ele tenta capturar o banner e com isso revelar qual o sistema operacional que o usuário final está usando.

cat p0f.log | less


Vamos olhar o TTL para comparar com alguns padrões de TTLs

Cyclades 	- 30
Linux 		- 64
Windows		- 128
Cisco		- 255
Linux + Netfilter - 255

vamos dar um ping no www.google.com

TTL = 56

o mais próximo é o do Linux 64 --> agora podemos fazer um traceroute para ver quantos saltos ouve para termos certeza

traceroute www.googole.com

nº de saltos =

TTL + Saltos = S.O.
56 + 8 = 64 = Linux

mtr www.google.com

existem formas de alterar o TTL então não é uma analise 100% real apenas uma ideia de qual o S.O. no alvo
