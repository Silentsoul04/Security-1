Comandos Avançados
==================

intitle, allintitle --> Busca as strings no titulo da página. eles ficam entre as tags <title></title>. a diferenças entre os dois comandos é que o primeiro busca apenas a primeira string no titulo, e a seguinda pegarai todas as strings após o comando. allintitle:index of | intile:index+of.

Exemplo:
	intitle:index.of

de uma navegada e mostre as informações que podemos obter atraves desse comando.

----

inurl, allinurl --> Busca strings na url. A diferença entre os comandos é semelhante ao intitle ,porém a busca é feita no endereço.


Exemplos:

vamos pesquisar possíveis acessos de VNC.
	intitle:VNC inurl:5800
	"VNC desktop" inurl:5800
	intitle:VNC inurl:5900

Vamos pesquisar Windows Small Business
	inurl:ConnectComputer/precheck.htm
	inurl:Remote/logon.aspx

Vamos buscar TS do windows Web Abero:
	intitle:Remote+Desktop inurl:tsweb
	intitle:Remote+Desktop inurl:tsweb intext:Domain


filetype --> Procura por aqruivos com a extensão procurada. caso vc precise consultar extensões pode ser feito através do site: http://filext.org.

Exemplo:
	quais extensões são legais de pesaquisar? sql, bak, bkp, gz, bz2, dump, doc/docx, xls/xlsx, pdf, dat, bdf,etc...

	filetype:sql
	filetype:sql inurl:gov.br
	

allintext --> Filtra conteúdos nas tags <body></body> e é semelhante a usar as strings pesquisadas dentro de "".

site --> faz a busca somente no site informado.

exemplo:
	filetype:sql site:agenersa.rj.gov.br

link --> busca por sites que contenha links para suas pesquisas.
	link:www.oys.com.br

inanchor --> encontra as strings dentro de uma ancora que são muito usados em páginas muito grande (#top).

datarange --> procura sites publicados entre as datas. e como parametro de pesquisa deverá ser feito com o calendário juliano, então tem que converter datas em nosso calendario para calendario juliano.

cache --> mostra a ultima versão da pág

info --> Procurar por string dentro do texto descritivo do site.

related --> é semelhante ao site, porém ele faz a busca por sites relacionados, que tenha o mesmo assunto,

Exemplo:
	related:www.uninove.br
